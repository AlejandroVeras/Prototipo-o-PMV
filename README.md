# InventoryAI - MVP

SaaS de Automatización de Inventarios con Visión Artificial

## 🎯 Descripción

InventoryAI es una solución innovadora para PYMES comerciales e industriales que automatiza la gestión de inventarios usando visión artificial. Reduce errores, ahorra tiempo y proporciona visibilidad en tiempo real.

## ✨ Características

- **Reconocimiento Visual**: Detecta y cuenta productos automáticamente usando cámaras estándar
- **Actualizaciones en Tiempo Real**: Sincronización automática de inventario
- **Reportes Automáticos**: Análisis detallado sin conteos manuales
- **Integración con ERP**: Compatible con sistemas contables locales
- **App Móvil**: Acceso desde cualquier lugar
- **Seguridad de Datos**: Encriptación de nivel empresarial

## 📊 Beneficios Clave

- **98% Precisión**: En conteos automáticos
- **5+ horas ahorradas por semana**: Por almacén
- **RD$12,000 pérdidas evitadas/mes**: En promedio por empresa
- **Cero equipos especiales**: Funciona con cámaras estándar

### Opción 2: Clonar Localmente
```bash
# Clonar repositorio
git clone https://github.com/[TU_USUARIO]/inventoryai-mvp.git
cd inventoryai-mvp

# Abrir en navegador (no requiere servidor)
```

### Opción 3: Ejecutar con servidor local
```bash
# Con Python 3

# Con Node.js (http-server)
```
